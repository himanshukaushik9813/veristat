One-sentence: Solid, opaque letter-grade block (A+…F) in bold mono — grades are unambiguous.

```jsx
<GradeChip grade="A" />
<GradeChip grade="C-" />
<GradeChip grade="F" size="lg" />
```

Sizes `sm` / `md` (default) / `lg`. Bands: A green, B blue-violet, C amber (dark ink), D orange (dark ink), F red. Helper `gradeBand(grade)` returns the band letter.
